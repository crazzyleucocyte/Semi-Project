주제 : 반려동물과 같이 산책 또는 여행할 장소 추천해주는 홈페이지

기획의도 :
1. 반려동물을 키우는 사람이 점점 많아지는 시대에 반려동물과 같이 놀러 갈 수 있는 곳을 찾기 쉽지 않다
	2. 전국에 있는 산책로, 반려동물동반 가능 문화시설의 정보와 함께 날씨와 미세먼지 정보를 제공하여 편의성을 높인다

요구사항 : 
- 로그인(회원, 관리자)
	- 회원가입
	- 마이페이지
	    1. 본인 정보 확인
	    2. 개인정보 변경(비밀번호 확인후 이동)
	        * 비밀번호 변경
		   # 기존 비밀번호 입력 후 이동
		   # 비밀번호 두 번 입력하여 변경
		* 기타 정보 변경
- 반려동물동반 문화시설 검색
	    1. 여러가지 검색 옵션 설정
	        * (카테고리별 검색, 지역별 검색, 키워드 검색 등등)
	    2. 클릭시 detailView
	        * 세부 사항(영업시간, 홈페이지 링크)
	        * 좋아요 기능(좋아요를 누르면 빨간 하트, 안누르면 빈 하트)
	        * 후기 남기기(본인 후기만 수정할 수 있게)
	        * 날씨 보기 클릭시 페이지 이동
	    3. 페이징처리
		
- 산책로 검색
	    1. 여러가지 검색 옵션 설정
	        * (카테고리별 검색, 지역별 검색, 키워드 검색 등등)
	    2. 클릭시 detailView
	        * 세부 사항(영업시간, 홈페이지 링크)
	        * 좋아요 기능(좋아요를 누르면 빨간 하트, 안누르면 빈 하트)
	        * 후기 남기기(본인 후기만 수정할 수 있게)
	        * 날씨 보기 클릭시 페이지 이동
	    3. 페이징처리
- 미세먼지와 날씨 표시 
	    1. 미세먼지(대기질 예보통보 조회)
	        * api 쿼리 결과 아래와 같은 데이터 반환
	        * 지도에 지역별로 표시(좋음, 보통, 나쁨, 매우나쁨 4단계로 4가지 색 이미지로 표시)(불가능할 경우 table로 표시)
	        * 서울 : 좋음, 제주 : 좋음, 전남 : 보통, 전북 : 보통,
	          광주 : 보통, 경남 : 좋음, 경북 : 좋음, 울산 : 좋음, 
	          대구 : 좋음, 부산 : 좋음, 충남 : 보통, 충북 : 좋음,
	          세종 : 보통, 대전 : 좋음, 영동 : 좋음, 영서 : 좋음, 
	          경기남부 : 좋음, 경기북부 : 좋음, 인천 : 보통
	    2. 날씨(기온, 강수확률, 날씨 표시)
	        * 산책로, 문화시설에서 클릭한 주소 값으로 api반환 결과 표시
	        * 지역별 검색
	        * 오늘 이후 1주일간의 날씨 변화
		   # N일 후 오전과 오후의 강수확률과 날씨를 테이블로 반환
